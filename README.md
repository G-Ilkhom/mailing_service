# Сервис управления рассылками

Программа по управлению рассылками, пользователями и сообщениями, а также получение статистики по проведённым рассылкам.
Рассылки отправляется автоматически по расписанию с использованием планировщика задач. 
Сервис предназначен для использования компаниями с целью информирования и привлечения клиентов.

## Основной функционал

- создание, редактирование и удаление рассылок
- просмотр списка рассылок
- автоматическая отправка сообщений по расписанию
- просмотр отчетов по проведённым рассылкам
- управление пользователями:
  - создание
  - редактирование
  - удаление
- администрирование через веб-интерфейс

## Установка и запуск проекта

### 1. Клонирование репозитория
```bash
git clone https://github.com/G-Ilkhom/mailing_service.git
cd mailing_service
```

### 2. Создание виртуального окружения
```bash
python -m venv venv
```

### 3. Активация виртуального окружения
Windows
```bash
venv\Scripts\activate
```
Linux / macOS
```bash
source venv/bin/activate
```

### 4. Установка зависимостей
```bash
poetry install
```

### 5. Настройка переменных окружения
Создайте файл .env на основе .env.sample и заполните необходимые значения.

### 6. Применение миграций
```bash
python manage.py migrate
```

### 7. Запуск
```bash
python manage.py runserver
```
